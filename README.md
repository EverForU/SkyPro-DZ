# Домашка SkyPro.

## Описание:
Банковский виджет для маскирования номера карты и номера счёта.

## Установка:
На данном этапе возможно использование через интегрированную среду разработки (IDE).

###Клонируйте репозиторий:
git clone https://github.com/

## Использование:

### Пример для карты
Visa Platinum 7000792289606361  # входной аргумент
Visa Platinum 7000 79** **** 6361  # выход функции

### Пример для счета
Счет 73654108430135874305  # входной аргумент
Счет **4305  # выход функции
